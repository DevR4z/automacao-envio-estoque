Script de automação para o ERP Olist em Python que utiliza PyAutoGUI para agilizar o envio de estoque para múltiplos marketplaces (Shopee, Mercado Livre, Amazon, Magalu e Shein). O projeto lê códigos de um arquivo CSV, interage dinamicamente com a interface gráfica e automatiza tarefas repetitivas de forma segura, visando otimizar o fluxo de gestão de estoque em e-commerce.
Principais tecnologias: Python, PyAutoGUI, CSV, automação desktop, RPA.
Foco: produtividade, integração e automação de rotinas.
